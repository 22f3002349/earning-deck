Code Sample: Simple NPV in Python


import math

def npv(rate, cashflows):
    """Compute Net Present Value.

    rate: discount rate as decimal (e.g., 0.1 for 10%)

    cashflows: list of cash flows where cashflows[0] is initial outlay (negative)
    """
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows))

# Example:
r = 0.12
cfs = [-1000, 300, 350, 400, 500]
print(npv(r, cfs))

Financial Equations

Inline NPV: \( \mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \)

Block CAGR:

\[ \mathrm{CAGR} = \left(\frac{V_{\text{final}}}{V_{\text{initial}}}\right)^{\tfrac{1}{n}} - 1 \]
